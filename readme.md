# 프로그램이 하는 일

exe파일이 있는 위치로 부터 FILE_EXE_PATH 위치의 파일이 있으면 실행하고 없으면 상위폴더로 이동해서 같은 작업을 반복한다.

0. 현재 폴더로부터 FILE_EXE_PATH 위치의 파일 검색
2. 파일이 있으면 실행
```
cmd> "해당 파일.exe" "현재 폴더(launcher exe 실행 위치)"
```
3. 없으면 상위 폴더로 이동하여 0번부터 반복
4. root(드라이브) 폴더이면 종료

### 프로젝트 수정

1. 빈 프로젝트 생성

2. WIN 프로그램 (창 숨기기)
  - 프로그램 속성 > Linker > System > Subsystem : Console로 되어 있으면 Window로 바꿔줘야함
